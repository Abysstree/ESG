# ESG 浏览器扩展 MVP

这个扩展用于把当前浏览器页面中已经显示出来的岗位正文导入本地 ESG。

## 使用方式

1. 启动本地后端：`cd backend && uv run uvicorn app.main:app --reload`。
2. 打开 Chrome 或 Edge 的扩展管理页。
3. 开启开发者模式。
4. 选择“加载已解压的扩展”，目录选择 `D:\abyss\Projects\ESG\browser-extension`。
5. 打开招聘岗位详情页。
6. 点击扩展图标，确认本地后端地址是 `http://127.0.0.1:8000`。
7. 在“导入正文预览”里检查并修改正文。
8. 点击“导入到 ESG”。

如果页面正文里混有很多导航、推荐岗位或广告，可以先在网页里选中岗位正文，再点击扩展按钮。扩展会优先导入选中的文本。

## 已支持的读取方式

- 选中文本优先：适合任何招聘页。
- BOSS 直聘页面清洗：对已经显示出来的 BOSS 岗位页 DOM 做基础噪声过滤。
- 页面可见文本回退：当没有选中文本或专用清洗失败时使用。

Popup 里的正文预览始终可以手动编辑，导入时以后端收到的预览框内容为准。

## 边界

- 扩展只读取当前浏览器页面中已经可见的 DOM 文本。
- BOSS 清洗只处理可见 DOM，不请求隐藏接口。
- 扩展不会破解接口、绕过登录、绕过验证码或模拟平台反爬参数。
- 扩展不会把 cookie 或 token 发给 ESG 后端。
